- teensy prop shield
- Taranis Transmitter/ reciver
- Propeller: carbon 13.5x5.5
- Motor: turnigy 3508-700

UAV specs:

- Configuration: hex x copter
- Weight without batteries: 2000g
- Arm length: 400mm
